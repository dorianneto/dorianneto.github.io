E agora?

Agora negada, com a modelagem de dados e o mapa mental finalizados, irei começar o desenvolvimento do nosso grande sistema de Tickets. Para que tudo seja feito de forma organizada, irei definir as prioridades de desenvolvimento:

  • Configuração inicial do framework
    • Aplicação
    • Banco de dados
    • Estruturação das views
    • Escolha das bibliotecas utilizadas
    • etc
  • Módulos auxiliares
  • Autenticação no sistema e controle de permissões
  • Módulos do sistema

Acho que assim conseguimos seguir de uma forma bem tranquila. Lembrando que todo o código desenvolvido será versionado e poderá ser acessado através do repositório no github. No próximo post irei colocar o link do repositório.
